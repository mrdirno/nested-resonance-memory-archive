### Citation Style Options

**Option 1: Nature Style (Numbered, Order of Appearance)**
```
Population homeostasis emerges through energy-regulated spawning dynamics¹.
Multi-scale validation²⁻⁴ reveals timescale-dependent patterns.
```

**Option 2: PLOS ONE Style (Numbered, Order of Appearance, Square Brackets)**
```
Population homeostasis emerges through energy-regulated spawning dynamics [1].
Multi-scale validation [2-4] reveals timescale-dependent patterns.
```

**Option 3: Ecology Style (Author-Year, Alphabetical)**
```
Population homeostasis emerges through energy-regulated spawning dynamics
(Payopay & Claude 2024a). Multi-scale validation (Levin 1992, Simon 1962,
Holling 1973) reveals timescale-dependent patterns.
```

**Recommendation:** PLOS ONE style (numbered, square brackets) - most common in computational/systems biology, matches target journal format.
