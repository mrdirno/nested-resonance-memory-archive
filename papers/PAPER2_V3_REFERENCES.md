## Citation Style Notes (PLOS Computational Biology)

**In-text citations:**
- Numbered sequentially in order of appearance: [1], [2], [3]
- Multiple citations: [1-3] or [1,2,3]
- Cite at end of sentence before period

**Reference formatting:**
- Journal articles: Authors (Year). Title. *Journal*, volume(issue), pages. DOI.
- Books: Authors (Year). *Title* (edition). Publisher, City. ISBN.
- Chapters: Authors (Year). Chapter title. In: Editor (Ed.), *Book Title*, pages. Publisher, City.

**Author formatting:**
- List all authors if ≤6
- If >6: First 3 authors, et al.
- Use initials for first/middle names
- Separate authors with commas
- Use & before last author

**PLOS-specific requirements:**
- DOI preferred over URL
- ISBN for books
- Full journal titles (no abbreviations)
- Include issue numbers in parentheses
- Pages: Use en dash (123-456, not 123–456)
