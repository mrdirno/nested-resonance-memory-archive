# Project Analysis for DUALITY-ZERO-V2

This folder contains a curated selection of files from the DUALITY-ZERO-V2 project, intended to provide a comprehensive overview for due diligence by another AI.

## Core Mandates and Objectives

*   `META_OBJECTIVES.md`: The main project tracking document, outlining current goals, status of papers, and recent progress.
*   `CLAUDE.md`: The project's "Constitution," detailing its core principles and mandates.
*   `README.md`: The top-level README for the entire project.

## Key Documentation

*   `MOG_Custom_Instructions_v2.md`: Describes the MOG-NRM integration and the "Living Epistemology Architecture."
*   `STEWARDSHIP_HELIOS_ARC_ROADMAP.md`: The long-term vision and roadmap for the project.
*   `REPRODUCIBILITY_GUIDE.md`: Instructions for reproducing the project's experiments and findings.

## Papers

The `papers/` directory contains all manuscripts, drafts, and submission packages. The most important, submission-ready papers are listed in `META_OBJECTIVES.md` and the main `README.md`.

## Key Analysis and Scripts

The `analysis/` directory contains a selection of key analysis scripts:
*   `realtime_emergence_analysis.py`: Analyzes massive-scale NRM dynamics from running experiments.
*   `pattern_archaeologist.py`: Implements "Pattern Archaeology" to trace the lineage and relationships of discovered patterns.
*   `aggregate_v6a_results.py`: An example of a script used to aggregate and analyze experimental results.

## Core Code Modules

The `code/` directory contains a selection of core Python modules that define the NRM framework:
*   `memory/pattern_memory.py`: The memory persistence layer for storing patterns, agent states, and historical data.
*   `fractal/fractal_swarm.py`: The multi-agent orchestrator implementing NRM dynamics.
*   `fractal/fractal_agent.py`: The core fractal agent definition.
