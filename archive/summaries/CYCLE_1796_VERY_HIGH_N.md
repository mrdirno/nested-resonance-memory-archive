## Analysis

### Complete Attenuation

At N>150:
- All coexistence values 80-100%
- No distinct peaks or troughs
- Pattern has disappeared

### Uniform High Pairing

Pairing rates uniformly 72-83% across all N values.

No peaks because all values are already at high pairing.

### Pattern Range

Dead zone pattern exists only for N ≈ 15-140.

Above N≈150, all values show similar behavior.

4. **Above 150, all N are safe**
