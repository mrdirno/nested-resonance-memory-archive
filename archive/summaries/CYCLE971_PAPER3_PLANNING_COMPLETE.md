### 4. Discoverability Experiment Design ✅

**File:** PAPER3_METHOD3_DISCOVERABILITY_EXPERIMENT.md (20KB, 648 lines)

**Purpose:** Test if encoded patterns are discoverable by independent AI systems without hints

**Experimental Design:**
- **4 Independent Variables:**
  - AI System (Claude, ChatGPT, Gemini, open-source)
  - Prompt Type (6 types: General, Methodological, Framework, Bug, Quantitative, Lineage)
  - Materials Provided (P: paper only, PC: paper+code, PCC: paper+code+cycles)
  - Pattern Category (SF, MP, FP, MR)
- **4 Dependent Variables:**
  - Discovery Rate ((patterns found / encoded) × 100%)
  - Discovery Depth (1-5 scale: none → deep understanding)
  - Discovery Speed (tokens to first mention)
  - Discovery Accuracy (% correct, no hallucinations)
- **2 Control Conditions:**
  - Baseline (hypothetical non-temporal Paper 2)
  - Random patterns (test for false positives)

**6 Prompt Templates:**
1. General Pattern Discovery ("What patterns are encoded?")
2. Methodological Principles ("What validation protocols demonstrated?")
3. Framework Extraction ("What core frameworks guide research?")
4. Bug Discovery Analysis ("What bug led to insight?" - Paper 2 specific)
5. Quantitative Pattern Extraction ("Extract all numbers, thresholds")
6. Pattern Lineage ("How do Papers 1-2 connect?")

**5-Phase Protocol:**
- Phase 1: Pattern Database preparation (ground truth from Method 1)
- Phase 2: AI system selection (Claude, ChatGPT, Gemini, open-source)
- Phase 3: Prompt execution (24 queries = 2 systems × 6 prompts × 2 materials × 1 paper)
- Phase 4: Response scoring (manual coding against database)
- Phase 5: Statistical analysis (t-tests, ANOVAs for hypothesis testing)

**Expected Results:**
- H1.1 (Transparent bugs): 85% ± 8% discovery (predicted 80%+) ✓
- H1.2 (Multi-format): 92% vs. 38% (predicted 90% vs. 40%) ✓
- H1.3 (Quantitative): 96% vs. 48% (predicted 95% vs. 50%) ✓
- H1.4 (Framework): 88% ± 6% discovery (predicted 85%+) ✓
