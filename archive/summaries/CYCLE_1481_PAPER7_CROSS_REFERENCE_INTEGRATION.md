**Observed Pattern:**
- 0-5 matches: No integration opportunities (Papers 3, 5D)

**Predictive Rule:**
- **<5 matches:** Unlikely (orthogonal topics)

- **>50 matches:** Very strong candidate (core topic overlap)

**Paper 7 Validation:**

- Actual result: 2 successful integrations
- Prediction confirmed ✅

### 2. Identified Gaps → Best Integration Points

**Paper 2 Integration:**
- **Context:** Future work question ("How does frequency interact with energy?")
- **Integration type:** Exploratory ("may extend to")
- **Strength:** Moderate (answers open question)

**Paper 7 Integration:**
- **Context:** Identified model limitation ("doesn't capture frequency-dependent variance")
- **Integration type:** Solution-oriented ("could inform", "address this limitation")

**Pattern:**
Papers explicitly stating limitations/gaps offer **stronger integration opportunities** than papers asking exploratory questions. Unified framework positioned as:
- **Paper 2:** Related work extending existing findings

### 3. Theoretical Papers Benefit More from Scaling Frameworks

**Paper 2 (Energy Dynamics):**
- Empirical observations → integration adds related empirical work
- Value: Cross-validation between studies

**Paper 7 (Governing Equations):**
- Mathematical formalization → integration adds empirical constraints
- Value: Guides theoretical refinement with data-driven scaling laws

**Lesson:** Theoretical/mathematical papers have higher integration value for empirical scaling frameworks. Paper 7's V3 parameter estimation could directly incorporate σ² ∝ f^-γ, while Paper 2 can only note the related finding.

### 4. Multi-Point Integration Appropriate for High-Match Papers

**Decision:** Paper 7 received 2 integrations (Discussion + Conclusions) vs Paper 2's 1 (Future Directions)

**Rationale:**
- **Match count similar:** 60 vs 62 (within 3% of each other)
- **Paper type different:** Theoretical framework vs empirical study
- **Gap clarity different:** Explicit limitation vs exploratory question
- **Value proposition different:** Actionable fix vs related work

**Rule of Thumb:**
- Papers with >50 matches **can support** multiple integration points if:
  - Each integration serves different purpose (problem statement vs future work)
  - Total additions remain concise (<100 words)
  - Flow not disrupted

**Counter-example:** Don't add 2 integrations just because match count is high—must have distinct, natural fit points.
